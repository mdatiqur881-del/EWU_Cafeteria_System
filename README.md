# EWU Cafeteria System

**EWU Cafeteria Management System** is a comprehensive C-based console application designed to streamline cafeteria operations at East West University. The system supports role-based access and provides features for employees, managers, customers, and faculty/staff, covering menu management, order handling, discounts, table management, and customer feedback.  

---

## Features

- **Role-based Access:**
  - Admin
  - Manager
  - Customer
  - Faculty/Staff

- **Employee Management:**
  - Add, view, and manage employee records

- **Food & Menu Management:**
  - Add and update food items
  - Maintain cafeteria menu
  - Track food reviews and ratings

- **Order Handling:**
  - Place and manage customer orders
  - Track table availability
  - Apply discounts automatically

- **Discount Management:**
  - Add and manage discount offers
  - Role-based usage of discounts

- **Customer Feedback:**
  - Collect reviews and ratings for food items

- **File-based Data Persistence:**
  - Employee, food, table, orders, and review data saved and loaded from text files
